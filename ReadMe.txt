Input
W,A,S,D - Forward, Left, Back and Right movement for the roman solider
Q,E - Up and Down movement for the roman solider
Up,Down arrow keys - zoom in, and out
Left,Right arrow keys - move camera left and right following a orbital path
R - Toggles automatic movement for the camera
1 - Toggles cursor
2 - Toggles Wireframe Mode
3 - Prints the mouse position on the screen
Mouse Left Click - when hovering over a button it will select the button
